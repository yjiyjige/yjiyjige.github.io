Sublime Text 入门

发表于 2015-06-07   |   分类于 Coding   |  

快捷键

一般的快捷键可以分为以下几类：

  • 编辑（Edit）
  • 选择（Selection）
  • 跳转（Go to）
  • 查找（Find）
  • 标签（Tabs）
  • 视图（View）
  • 工程（Project）
  • 通用（General）

编辑（Edit）

  • CMD + [、] 改变缩进
  • CMD + CTRL + 上、下 交换行位置
  • CMD + SHIFT + D 复制与粘贴
  • CMD + 左、右 行开头与结尾
  • CMD + / 注释
  • CMD + CTRL + / 块注释
  • CTRL + K 从光标删除到行尾
  • CTRL + SHIFT + K 删除行，CMD + X有类似效果
  • CMD + OPTION + T 插入特殊符号

光标选中（Selection）

  • CMD + D 选中相同的词
  • CMD + CTRL + G 多重文本选中
  • OPTION + 鼠标选择或鼠标中键选择 块选择
  • CTRL + SHIFT + M 选中当前匹配括号中的内容

查找（Find）

  • CMD + F 查找
  • CMD + OPTION + F 替换
  • CMD + G 查找下一处
  • CMD + SHIFT + F 从文件在查找
  • CMD + E 缓存用于查找的内容
  • CMD + SHIFT + E 缓存用于替换的内容
  • CMD + OPTION + E 替换下一个

跳转（Go to）

  • CMD + P 文件跳转
  • CMD + R 定位文件中的方法
  • CMD + G 定位行
  • CTRL + M 定位匹配括号
  • F2 下一个书签
  • CMD + F2 开关书签
  • CMD + SHIFT + F2 清除所有书签
  • CMD + OPTION + 左、右 打开文件跳转

通用

  • CMD + SHIFT + P 命令
  • CMD + K,CMD + B 开关侧边栏

标签（Tabs）

  • CMD + SHIFT + T 最后关闭的tab
  • CTRL + TAB tab之间切换
  • CMD + 数字　切到相应的tab
